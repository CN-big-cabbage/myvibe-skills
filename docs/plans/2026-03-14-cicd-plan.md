# CI/CD Pipeline Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add CI/CD to myvibe-skills with PR checks (ESLint + Prettier + vitest + security scan) and auto/manual release workflow.

**Architecture:** Two GitHub Actions workflows. `ci.yml` runs on every PR and push to main for quality gates. `release.yml` runs on push to main (auto) and workflow_dispatch (manual) for versioning and releases. ESLint + Prettier enforce code style, configured at project root targeting `scripts/**/*.mjs`.

**Tech Stack:** GitHub Actions, ESLint (flat config), Prettier, Vitest, npm audit, TruffleHog

### Task 6: Create CI workflow

**Files:**
- Create: `.github/workflows/ci.yml`

**Step 1: Create directory**

```bash
mkdir -p .github/workflows
```

**Step 2: Create `.github/workflows/ci.yml`**

```yaml
name: CI

on:
  pull_request:
    branches: [main]
  push:
    branches: [main]

jobs:
  lint-and-test:
    name: Lint & Test
    runs-on: ubuntu-latest

    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: 20

      - name: Install dependencies
        run: npm install --prefix skills/myvibe-publish/scripts

      - name: Lint
        run: npm run lint --prefix skills/myvibe-publish/scripts

      - name: Format check
        run: npm run format --prefix skills/myvibe-publish/scripts

      - name: Test
        run: npm test --prefix skills/myvibe-publish/scripts

  security:
    name: Security Scan
    runs-on: ubuntu-latest

    steps:
      - name: Checkout
        uses: actions/checkout@v4
        with:
          fetch-depth: 0

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: 20

      - name: Install dependencies
        run: npm install --prefix skills/myvibe-publish/scripts

      - name: Dependency audit
        run: npm audit --prefix skills/myvibe-publish/scripts --audit-level=high
        continue-on-error: true

      - name: Secret scan
        uses: trufflesecurity/trufflehog@main
        with:
          extra_args: --only-verified
```

### Task 7: Create Release workflow

**Files:**
- Create: `.github/workflows/release.yml`

**Step 1: Create `.github/workflows/release.yml`**

```yaml
name: Release

on:
  push:
    branches: [main]
  workflow_dispatch:
    inputs:
      bump:
        description: 'Version bump type'
        required: true
        type: choice
        options:
          - patch
          - minor
          - major

permissions:
  contents: write

jobs:
  release:
    name: Release
    runs-on: ubuntu-latest
    # Skip if commit message starts with 'chore(release)'
    if: "!startsWith(github.event.head_commit.message, 'chore(release)')"

    steps:
      - name: Checkout
        uses: actions/checkout@v4
        with:
          fetch-depth: 0
          token: ${{ secrets.GITHUB_TOKEN }}

      - name: Configure git
        run: |
          git config user.name "github-actions[bot]"
          git config user.email "github-actions[bot]@users.noreply.github.com"

      - name: Determine bump type
        id: bump
        run: |
          if [ "${{ github.event_name }}" = "workflow_dispatch" ]; then
            echo "type=${{ github.event.inputs.bump }}" >> "$GITHUB_OUTPUT"
            echo "skip=false" >> "$GITHUB_OUTPUT"
            exit 0
          fi

          # Auto mode: parse commits since last tag
          LAST_TAG=$(git describe --tags --abbrev=0 2>/dev/null || echo "")

          if [ -z "$LAST_TAG" ]; then
            RANGE="HEAD"
          else
            RANGE="${LAST_TAG}..HEAD"
          fi

          COMMITS=$(git log "$RANGE" --pretty=format:"%s" 2>/dev/null || echo "")

          if echo "$COMMITS" | grep -qiE "BREAKING CHANGE|^[a-z]+(\(.+\))?!:"; then
            echo "type=major" >> "$GITHUB_OUTPUT"
          elif echo "$COMMITS" | grep -qE "^feat(\(.+\))?:"; then
            echo "type=minor" >> "$GITHUB_OUTPUT"
          elif echo "$COMMITS" | grep -qE "^(fix|perf)(\(.+\))?:"; then
            echo "type=patch" >> "$GITHUB_OUTPUT"
          else
            echo "type=none" >> "$GITHUB_OUTPUT"
            echo "skip=true" >> "$GITHUB_OUTPUT"
            exit 0
          fi

          echo "skip=false" >> "$GITHUB_OUTPUT"

      - name: Skip release
        if: steps.bump.outputs.skip == 'true'
        run: echo "No releasable commits found. Skipping release."

      - name: Calculate new version
        if: steps.bump.outputs.skip != 'true'
        id: version
        run: |
          LAST_TAG=$(git describe --tags --abbrev=0 2>/dev/null || echo "v0.0.0")
          CURRENT="${LAST_TAG#v}"

          IFS='.' read -r MAJOR MINOR PATCH <<< "$CURRENT"

          case "${{ steps.bump.outputs.type }}" in
            major) MAJOR=$((MAJOR + 1)); MINOR=0; PATCH=0 ;;
            minor) MINOR=$((MINOR + 1)); PATCH=0 ;;
            patch) PATCH=$((PATCH + 1)) ;;
          esac

          NEW_VERSION="${MAJOR}.${MINOR}.${PATCH}"
          echo "version=$NEW_VERSION" >> "$GITHUB_OUTPUT"
          echo "New version: $NEW_VERSION"

      - name: Update version files
        if: steps.bump.outputs.skip != 'true'
        run: |
          VERSION="${{ steps.version.outputs.version }}"

          # Update .claude-plugin/marketplace.json
          node -e "
            const fs = require('fs');
            const f = '.claude-plugin/marketplace.json';
            const data = JSON.parse(fs.readFileSync(f, 'utf8'));
            data.metadata.version = '${VERSION}';
            fs.writeFileSync(f, JSON.stringify(data, null, 2) + '\n');
          "

          # Update scripts/package.json
          node -e "
            const fs = require('fs');
            const f = 'skills/myvibe-publish/scripts/package.json';
            const data = JSON.parse(fs.readFileSync(f, 'utf8'));
            data.version = '${VERSION}';
            fs.writeFileSync(f, JSON.stringify(data, null, 2) + '\n');
          "

      - name: Generate changelog
        if: steps.bump.outputs.skip != 'true'
        id: changelog
        run: |
          LAST_TAG=$(git describe --tags --abbrev=0 2>/dev/null || echo "")

          if [ -z "$LAST_TAG" ]; then
            RANGE="HEAD"
          else
            RANGE="${LAST_TAG}..HEAD"
          fi

          {
            echo "changelog<<CHANGELOG_EOF"

            FEATS=$(git log "$RANGE" --pretty=format:"- %s" --grep="^feat" --extended-regexp 2>/dev/null || echo "")
            if [ -n "$FEATS" ]; then
              echo "### Features"
              echo "$FEATS"
              echo ""
            fi

            FIXES=$(git log "$RANGE" --pretty=format:"- %s" --grep="^fix" --extended-regexp 2>/dev/null || echo "")
            if [ -n "$FIXES" ]; then
              echo "### Bug Fixes"
              echo "$FIXES"
              echo ""
            fi

            OTHERS=$(git log "$RANGE" --pretty=format:"- %s" --grep="^feat\|^fix" --invert-grep --extended-regexp 2>/dev/null || echo "")
            if [ -n "$OTHERS" ]; then
              echo "### Other Changes"
              echo "$OTHERS"
              echo ""
            fi

            echo "CHANGELOG_EOF"
          } >> "$GITHUB_OUTPUT"

      - name: Commit, tag, and push
        if: steps.bump.outputs.skip != 'true'
        run: |
          VERSION="${{ steps.version.outputs.version }}"
          git add .claude-plugin/marketplace.json skills/myvibe-publish/scripts/package.json
          git commit -m "chore(release): v${VERSION}"
          git tag "v${VERSION}"
          git push origin main --follow-tags

      - name: Create GitHub Release
        if: steps.bump.outputs.skip != 'true'
        env:
          GH_TOKEN: ${{ secrets.GITHUB_TOKEN }}
        run: |
          VERSION="${{ steps.version.outputs.version }}"
          gh release create "v${VERSION}" \
            --title "v${VERSION}" \
            --notes "${{ steps.changelog.outputs.changelog }}"
```
